Purpose: Simple HTML module for OpenCart

Installation Notes:
1.Upload files to the corresponding directories.
2.From the Administration panel enter Modules from the Extensions menu.
3.Install mymodule module.
4.If you do not have permission you will have to set permission to modules/mymodule from the top administrator user group from within the admin configuration user menu.
5.Edit the MyModule settings for position of module.

Modification notes:
1. You can modify the content to your like, for example simple information or maybe displaying ads like in my website here
linux.kreasimote.com
2. Now support multilanguage for header and content
